# bo2-gsc-injector
BO2 1.19 GSC script injector for CFW PlayStation 3s.

## Parameters 
`args[0] : tm TM TMAPI cc CC CCAPI`

`args[1] : project_directory`

### Usage Example 
`bo2-gsc-injector.exe TM "/projects/gsc/test-project"`
